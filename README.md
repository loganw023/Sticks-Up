# Sticks-Up

This is a hardware-based fencing game made during the outpost hackathon. It features a platform where two robots fight against each other with various movement sets and a pair of controllers for players to control the robots. Here is a link to a Youtube short with our demo:  https://youtube.com/shorts/5DaKTgEOJ2I?feature=share
